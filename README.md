# Vitalijus Lazarevas — Portfolio

A modern personal portfolio website built to showcase my skills, projects, and contact information.

## Live Demo

https://cv-portfolio-vl.vercel.app

## Features

- **Spotlight Effect** — Interactive cursor-following glow on the hero section
- **Multi-Language Support** — English/Lithuanian toggle with saved preference in localStorage
- **Theme Toggle** — Dark/light mode switch with persistent theme state
- **Responsive Design** — Fully adaptive layout for desktop, tablet, and mobile
- **Project Cards** — Image previews with hover overlays, descriptions, tech stack, and links
- **CV Download** — Dropdown with English and Lithuanian CV versions
- **Contact Section** — LinkedIn, GitHub, and Email links in the About section
- **Smooth Navigation** — Fixed navbar with scroll-based styling and mobile hamburger menu

## Tech Stack

| Technology | Purpose |
|---|---|
| React | UI components and state management |
| Vite | Fast build tool and dev server |
| SCSS | Modular styling with variables and mixins |
| React Icons | SVG icon library |
| localStorage | Persisting theme and language preferences |

## Project Structure

- `src/components` — UI components
- `src/contexts` — Language context
- `src/data` — Translations, skills, projects, and about data
- `src/assets` — Images, logos, and CV files
